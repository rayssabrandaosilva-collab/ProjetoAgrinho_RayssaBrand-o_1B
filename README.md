# ProjetoAgrinho_RayssaBrand-o_1B
EcoVida – O Futuro Sustentável é um site educativo que incentiva a conscientização ambiental por meio de conteúdos interativos sobre sustentabilidade, preservação e reciclagem. O projeto inclui um quiz com 25 perguntas, certificado de conclusão e recursos de acessibilidade para oferecer uma experiência inclusiva.
